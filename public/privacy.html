Privacy Policy — ZAlgo

Last updated: 2026-02-05

Summary

ZAlgo is a Chrome extension for X (Twitter) that estimates engagement-based scores using publicly visible signals. All processing is performed locally in your browser.

No data is transmitted to any server.

Data Collected

  • Public tweet engagement metrics
  • Tweet metadata (timestamp, media type)
  • Calculated scores and breakdowns
  • Local history snapshots

How Data Is Used

Data is used solely to compute and display local analytics within the extension UI.

Storage

All data is stored using chrome.storage.local. No cloud sync or external storage is used.

Permissions

  • storage – local caching
  • unlimitedStorage – prevents silent eviction
  • Host access limited to x.com and twitter.com

Data Control

Users can clear or wipe all stored data at any time via the extension dashboard.

Third-Party Sharing

ZAlgo does not sell, share, or transfer data to third parties.

Children’s Privacy

This extension is not intended for children under 13.
