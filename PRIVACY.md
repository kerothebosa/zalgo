# Privacy Policy — ZAlgo

**Last updated:** 2026-02-05

## Summary

ZAlgo is a Chrome extension for X (Twitter) that estimates engagement-based scores
using **public engagement signals**.  
All processing is performed **locally** in your browser.

No data is transmitted to any server.

---

## Data Collected

ZAlgo processes and stores the following data **locally on your device**:

- Tweet metrics (likes, replies, reposts, bookmarks, views)
- Tweet metadata (timestamp, media type)
- Calculated score values and breakdowns
- Local history snapshots (capped per tweet)

---

## How Data Is Used

Data is used only to:
- Compute estimated scores
- Display UI overlays and dashboards
- Provide local analytics and history

---

## Storage

- All data is stored using `chrome.storage.local`
- No cloud sync
- No third-party access

---

## Permissions

- `storage` – required for local caching
- `unlimitedStorage` – prevents silent eviction for heavy users
- Host access limited to:
  - `https://x.com/*`
  - `https://twitter.com/*`

---

## Data Retention & Control

- No automatic time-based deletion
- Manual cleanup available inside the extension
- Users may wipe all stored data at any time

---

## Third-Party Sharing

❌ No data is sold, shared, or transferred.

---

## Children’s Privacy

This extension is not intended for users under 13.
