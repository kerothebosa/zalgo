ZAlgo

Engagement quality indicators derived from publicly visible interaction signals, calculated locally in your browser.

What ZAlgo Does

ZAlgo is a browser extension that displays small, unobtrusive engagement quality indicators alongside social content. These indicators help users understand the relative strength of visible interaction — beyond simple like or view counts.

ZAlgo does not modify content, automate actions, or influence what platforms show. It provides additional context based solely on information already visible on the page.

How It Works

ZAlgo evaluates publicly visible interaction signals such as replies, reposts, likes, bookmarks, views, and time since posting. These signals are normalized, weighted, and adjusted for recency to produce an estimated engagement quality indicator.

All calculations run entirely within the browser. No external services or servers are used.

Privacy & Data Handling

  • No accounts or sign-in required
  • No network requests or data uploads
  • No third-party analytics or tracking
  • All data stored locally using browser storage
  • Manual cleanup controls available within the extension

Code Transparency

Production builds of the extension use standard JavaScript minification and obfuscation solely to protect proprietary implementation details and reduce bundle size.

Important: Obfuscation is not used to load remote code, execute dynamic scripts, track users, or conceal behavior. All logic is static and bundled at build time.

Compatibility
