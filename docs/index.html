Browser Extension

ZAlgo

Engagement quality indicators derived from publicly visible interaction signals — calculated locally, transparently, and without tracking.

What ZAlgo Does

ZAlgo is a browser extension that displays unobtrusive engagement quality indicators next to social content. These indicators help interpret interaction strength beyond raw like or view counts.

ZAlgo does not automate actions, modify content, or influence platform behavior. It provides contextual insight using information already visible on the page.

How It Works

ZAlgo evaluates publicly visible interaction signals such as replies, reposts, likes, bookmarks, views, and time since posting. These signals are normalized and adjusted for recency to produce an estimated engagement quality indicator.

All computation happens locally inside the browser. No external services or servers are involved.

Privacy & Data Handling

  • No accounts or sign-in required
  • No network requests or data uploads
  • No third-party analytics or tracking
  • All data stored locally using browser storage
  • Manual cleanup controls available inside the extension

Code Transparency

Production builds use standard JavaScript minification and obfuscation solely to protect proprietary implementation details and reduce bundle size.

Obfuscation is not used to load remote code, execute dynamic scripts, track users, or conceal behavior. All logic is static and bundled at build time.

Compatibility & Branding

ZAlgo is compatible with x.com and twitter.com.
