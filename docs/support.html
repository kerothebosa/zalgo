ZAlgo Support

Help, issue reporting, and general information.

How to Get Help

ZAlgo is designed to be simple and self-contained. If you encounter issues, unexpected behavior, or have questions about how the extension works, you can reach support using the options below.

Report an Issue

If something does not work as expected (for example, indicators not appearing after a platform interface change), please include:

  • The page URL where the issue occurred
  • A short description of what you expected vs. what happened
  • Your browser version
Recommended:
Submit issues via the project’s GitHub repository issue tracker.

Privacy & Data Questions

ZAlgo does not transmit data externally. All processing and storage occur locally within the browser. If you have questions about data handling, storage cleanup, or permissions, please consult the privacy documentation linked from the store listing or contact support.

What Support Cannot Do

  • Access your browsing data
  • Recover deleted local data
  • Modify platform behavior or rankings
  • Provide official platform insights

Branding & Affiliation

ZAlgo is an independent project. Support communications do not represent X Corp., Twitter, or any affiliated organization.
