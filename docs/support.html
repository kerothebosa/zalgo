ZAlgo Support

Help, issue reporting, and product information.

Getting Help

ZAlgo is designed to be simple and self-contained. If you experience issues, unexpected behavior, or have questions about how the extension works, support is available using the options below.

Reporting Issues

When reporting an issue, please include:

  • The page URL where the issue occurred
  • A brief description of expected vs actual behavior
  • Your browser version
Preferred support channel
Please submit issues through the project’s GitHub issue tracker.

Privacy & Data Questions

ZAlgo performs all processing locally in the browser and does not transmit data externally. Questions related to data storage, permissions, or cleanup are welcome.

Support Limitations

  • No access to user browsing data
  • No recovery of deleted local data
  • No ability to modify platform behavior
  • No official platform insights or endorsements
